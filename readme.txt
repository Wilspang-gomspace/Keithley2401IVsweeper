This is a script and the resulting pyinstaller executable to perform automated IV sweeps on a Keithley 2401 source measurement unit. 

Installation prequisites:
1. install labview runtime environment. 
LVRTE2018_f2Patch-64std
2. install the national instruments 488.2 / SCPI driver for the GPIB.
NI4882_1700f0
3. install the national instruments serial package. 
niserial1400
4. install the national instruments visa interface
NIVISA1700full
